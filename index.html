Bootstrap Drawer drawer.js

Drawers are streamlined, but flexible, dialog prompt from left or right of the page with minimum required functionality and smart default. This plugin implement many of modal.js feature, in fact this addons is modified version of modal.js.

Usage

The drawer plugin toggles your hidden content on demand, via data attribute or JavaScript. It also add .drawer-open to the body to override default scrolling behavior and and generates a .drawer-backdrop to provide a click area for dismissing shown drawer when clicking outside the drawer.

Via data attributes

Activate a drawer without writing JavaScript. Set data-toggle="drawer" on a controller element, like a button, along with data-target="#foo" or href="#foo" to target a specific drawer to toggle.

<button type="button" data-toggle="drawer" data-target="#myDrawer">Launch drawer</button>

Via JavaScript

Call a drawer with id myDrawer with a single line of JavaScript.

$('#myDrawer').drawer(options)

Methods

.drawer(options)

Activate your content as drawer. Accept an optional option object.

$('#myDrawer').drawer({ show: true })

.drawer('toggle')

Manual toggles a drawer.

$('#myDrawer').drawer('toggle')

.drawer('show')

Manual opens a drawer.

$('#myDrawer').drawer('show')

.drawer('hide')

Manual hides a drawer.

$('#myDrawer').drawer('hide')

Events

Bootstrap drawer exposes a few events for hooking into drawer functionality.

All drawer events are fired at the drawer itself (i.e. at the <div class="drawer">).

Event TypeDescription
show.bs.drawer This event fires immediately when the show instance method is called. If caused by a click, the clicked element is available as the relatedTarget property of the event.
shown.bs.darwer This event is fired when the drawer has been made visible to the user (will wait for CSS transitions to complete). If caused by a click, the clicked element is available as the relatedTarget property of the event.
hide.bs.drawer This event is fired immediately when the hide instance method has been called.
hidden.bs.drawer This event is fired when the drawer has finished being hidden from the user (will wait for CSS transitions to complete). 
$('#myDrawer').on('hidden.bs.drawer', function(e){
    // do something
});
